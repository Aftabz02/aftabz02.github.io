Frequently Asked Questions

What is a refurbished computer or laptop?

A refurbished unit is a used system fully tested, cleaned and restored. Parts like batteries, keyboards or screens are replaced where needed. We install a fresh OS and run reliability checks before sale.

Are refurbished systems reliable?

Yes — we test thoroughly, and include a 30‑day hardware warranty for peace of mind.

How do I purchase?

Browse our eBay store and message us if you have questions. Local collection can be arranged.

Which payments do you take?

Online purchases go through eBay/PayPal. For repairs, we accept standard methods (ask for details).

What is your diagnostics fee?

It’s £20 and we waive it if you proceed with the repair.

Need help today?

Fast diagnostics, honest pricing, friendly service.
